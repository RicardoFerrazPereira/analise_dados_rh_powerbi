# 👥 Projeto de Análise de Dados de RH com Power BI

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com foco em análise de dados de Recursos Humanos utilizando o Microsoft Power BI.

O dashboard apresenta indicadores estratégicos relacionados aos colaboradores da empresa, permitindo análises sobre quadro de funcionários, salários, experiência profissional, envolvimento no trabalho e elegibilidade para promoções.

O objetivo é transformar dados de RH em informações visuais e acionáveis para apoiar decisões gerenciais.

---

# 🎯 Objetivos da Análise

O dashboard responde às seguintes perguntas de negócio:

* Qual o total de funcionários atualmente na empresa?
* Qual o tempo médio de experiência dos funcionários?
* Qual o total e percentual de funcionários por gênero?
* Qual a média salarial mensal?
* Qual o total de funcionários por função?
* Qual o percentual de funcionários disponíveis para hora extra?
* Qual o nível de envolvimento dos funcionários no trabalho?
* Qual o total e percentual de funcionários elegíveis para promoção?
